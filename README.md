# Instamaps

Eina de l’Institut Cartogràfic i Geològic de Catalunya (ICGC) posa a la teva disposició per tal que puguis explorar la geoinformació de Catalunya i el món, crear els teus mapes de manera ràpida i fàcil, visualitzar-los en qualsevol dispositiu i compartir-los amb les xarxes socials.
